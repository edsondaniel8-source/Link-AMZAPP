 Problema Identificado:
Frontend está chamando:

text
POST /api/rides-simple/create
Mas seu backend só tem:

javascript
app.post('/api/rides', ...) // ← Rota diferente!
🔧 Soluções Possíveis:
Opção 1: Alterar o Backend (Recomendado)
javascript
// Mude a rota no backend para coincidir com o frontend
app.post('/api/rides-simple/create', (req, res) => {
  try {
    const {
      type,           // ← Agora é 'type' em vez de 'vehicleType'
      fromAddress,    // ← Agora é 'fromAddress' em vez de 'from'  
      toAddress,      // ← Agora é 'toAddress' em vez de 'to'
      departureDate,  // ← Agora é 'departureDate' em vez de 'date'
      price,          // ← Agora é 'price' em vez de 'price'
      // ... outros campos
    } = req.body;

    console.log('📦 Dados recebidos:', req.body);

    // Sua lógica aqui...
    
    res.status(201).json({ success: true, message: 'Rota criada!' });

  } catch (error) {
    res.status(500).json({ error: error.message });
  }
});
